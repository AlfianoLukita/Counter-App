# Counter App

Simple counter web app made with HTML, CSS, and JavaScript.

## 🚀 Features
- Increase and decrease counter
- Dark/Light Theme
- Quick add buttons (+5)
- Quick subtract buttons (−5)
- Reset button
- Visual indicator of added/subtracted value

## 🛠️ Technologies
- HTML
- CSS
- JavaScript
